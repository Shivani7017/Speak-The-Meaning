# Voice Dictionary Assistant

A Python-based voice-enabled dictionary that fetches meanings of words and reads them aloud. Perfect for learning new words interactively.  

---

## Features

- Input a word via keyboard.  
- Fetch multiple meanings (noun, verb, adjective, etc.).  
- Speak the meaning aloud using text-to-speech.  
- Interactive and easy-to-use command-line interface.  

---

## Technologies Used

- Python 3.x  
- [PyDictionary](https://pypi.org/project/PyDictionary/)  
- [pyttsx3](https://pypi.org/project/pyttsx3/)
